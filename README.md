# 🎥 Noom

- 노마드코더 줌 클론코딩을 따라하며 만들어보는 레포지토리입니다.

- Zoom과 비슷하게 영상 및 음성처리까지 구현하는 것이 목표입니다.

- 강의의 0.0 ~ 3.2까지 구현되어 있습니다.

- 블로그에 7번째 글까지 포스팅되어 있습니다.
